# MathDarts
Простая игра под названием «математический дартс», суть которой в улучшении скорости счёта в уме. Реализовано только сложение целых чисел. Написана в рамках изучения языка Python и библиотеки Tkinter. 
Файл MathDarts.ipynb написан с использованием Jupyter Notebook.
Файл MathDarts.py является результатом экспорта кода из файла MathDarts.ipynb.
