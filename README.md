# MathDarts
Простая игра под названием "математический дартс" суть которого в улучшении скорости счёта в уме. Реализовано только сложение целых чисел. Написана в рамках изучения языка python и библиотеки TKinter. 
Файл MathDarts.ipynb написан с использование JupiterNotebook
Файл MathDarts.py является результатом экспорта кода из файла MathDarts.ipynb
